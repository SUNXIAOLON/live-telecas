## web直播demo

> web直播demo，包括直播、点播页面，实现的RTMP协议流媒体服务端

#### 准备环境

> node

#### 快速开始
``` bash
  cd live-telecas
  npm i

  // 启动流媒体服务端
  npm run service
  
  // 启动直播页面
  npm run live

  // 启动点播播页面
  npm run play
```
